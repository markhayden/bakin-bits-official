# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][cc].

In short: be respectful, assume good faith, and help keep this a welcoming
project for contributors of all backgrounds and experience levels.

## Reporting

If you experience or witness behavior that violates this code, please report
it privately to **hi@markhayden.me**. Reports are handled confidentially and
acknowledged within 72 hours.

When reporting, include what happened, where it happened (issue, PR, commit,
chat link), and any context that helps us understand the situation.

## Enforcement

Maintainers may take any action they deem appropriate, including warnings,
temporary bans, or permanent removal from the project. The Contributor
Covenant's [enforcement guidelines][cc-enforce] are the reference for what
proportionate action looks like.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[cc-enforce]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
